# SMARTINVENTORY-mini

A simple PHP inventory management system with product, supplier, and purchase tracking.

## Project structure

- `index.php` - main dashboard or entry point
- `config/db.php` - database connection setup
- `includes/header.php` - shared page header
- `includes/footer.php` - shared page footer
- `products/` - product management pages
- `purchases/` - purchase management pages
- `suppliers/` - supplier management pages
- `reports/low_stock.php` - low stock reporting

## Setup

1. Place this project in your PHP web server root.
2. Configure `config/db.php` with your database credentials.
3. Open the project in your browser.
